## Quick Start

Quick start option:

- Extract the zip file.
- Navigate inside the folder.
- `npm i material-dashboard-react`

###

## Run this if you face any issues:

- Run the following in command line or powershell:
- `npm cache clean --force `
- Navigate inside the folder.
- `npm install`
- `npm start`
